# Kelionių maršrutų valdymo sistema

# Projekto tikslas
Sukurti paprastą C++ programą, kuri leidžia saugoti ir valdyti kelionių duomenis.

# Programos aprašymas
Programa skirta kelionių informacijai tvarkyti. Vartotojas gali įvesti naujas keliones, jas redaguoti, ištrinti arba peržiūrėti visas esančias.

Kiekviena kelionė turi:
šalį
miestą
transporto tipą
kainą
trukmę dienomis

# Funkcionalumas

#  operacijos:
pridėti naują kelionę
peržiūrėti visas keliones
redaguoti pasirinktą kelionę
ištrinti kelionę

## Papildomos funkcijos:
paieška pagal šalį arba miestą
 pigiausios kelionės radimas

# Naudojami failai
main.cpp – programos kodas
keliones.txt – duomenų failas
